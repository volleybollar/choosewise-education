Reception class teacher

Five sample prompts from this prompt pack

Every prompt is a template with bracketed placeholders [replace this text] that you swap for your own context.

  1. Write a short guide for introducing a new [specific skill, e.g. writing their name] to a reception class (6-year-olds), with three steps and a playful method.
  2. Suggest three ways to collaborate with grade 1 teachers to ease the transition for a reception class (6-year-olds), focusing on [specific goal, e.g. reading], with one idea for each.
  3. Give me five ways to use nature with a reception class (6-year-olds) to practise [specific goal, e.g. counting], with one playful idea for each.
  4. Suggest three ways to collaborate with after-school staff for a reception class (6-year-olds) to practise [specific goal, e.g. cooperation], with one idea for each.
  5. Suggest three fun movement activities for a reception class (6-year-olds) that work as movement breaks when children are finding it hard to sit still.

How to use them

Paste the prompt into a chatbot. Where you see [bracketed text], replace it with what fits your situation. Always double-check output for accuracy — and never upload personal or sensitive data.

All prompts were originally developed in a Swedish school context. This means that some prompts may need to be adapted to the curriculum and context in your country.

These prompts pair with the WISE Framework for Education: the framework helps you decide whether to use an AI tool; the prompts help you use it well once you do.
